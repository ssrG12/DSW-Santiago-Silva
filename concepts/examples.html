Ejemplos:

Inline:

Este es un texto de ejemplo para explicar la propiedad

INLINE

Como puedes ver, el texto sigue su curso, no se altera, sino que sigue en el mismo flujo de pantalla. Si hacemos el texto un poco mas largo, puedes darte cuenta que ese elemento no genera ningún espacio adicional.

Block:

Este es un texto de ejemplo para explicar la propiedad

BLOCK

Como puedes ver, al aplicar la porpiedad BLOCK, la palabra toma ese bloque completo de la pantalla, y el resto de elementos continuan debajo de él. Además genera cierto espacio alrededor suyo.

Inline-block:

Este es un texto de ejemplo para explicar la propiedad

INLINE-BLOCK

Como puedes ver, todo el texto sigue su curso, pero al rededor de la palabra se forma un bloque específico, con espacio a los lados, pero sin tomar todo el ancho de la pantalla.
